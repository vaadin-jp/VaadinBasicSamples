# VaadinBasicSamples

[vaadin.jp](https://vaadin.jp) の公式リポジトリ（Vaadinの基本機能のサンプルコード集）です。  

[vaadin.jp](https://vaadin.jp) は、[vaadin.com](https://vaadin.com)と協力し、日本向けに情報を発信しています。

[vaadin.jp](https://vaadin.jp) は、Vaadin社の公式パートナー[SoftCommu]([https://vaadin.com](https://softcommu.com/))によって運営されています。

---

## 概要

このリポジトリでは、Vaadin フレームワークの基本的な使い方をサンプルコードを通して学ぶことができます。
Vaadinは、SpringBootと組み合わせて動作するフルスタック開発用ワークフレームです。
SpringBoot と Thymeleaf の組み合わせでは、フロントエンドコードのコード（HTML、CSS、JavaScript）のコーディングが必要になりますが、
SpringBoot と Vaadin の組み合わせでは、Java「だけ」でフロントエンドとバックエンドの開発が同時にできます。

対象者:

- SpringBoot + Vaadin をこれから学びたい方
- SpringBoot + Vaadin を日本語で学びたい方
- Java で Web アプリを構築したい方
- UI コンポーネントの基本操作を確認したい方

---

## 動作環境

- Java 17 以上
- Gradle 7 以上
- ブラウザ（Chrome / Edge / Firefox など）

---

## アプリの起動方法

### 1. リポジトリをクローン

```bash
git clone https://github.com/vaadin-jp/VaadinBasicSamples.git
```

### 2. IDEに gradleプロジェクトとして取り込む。

Eclipse、VS Code、InteliJ IDEA などに取り込んでください。
